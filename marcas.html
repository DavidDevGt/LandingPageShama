Marcas

Ferretería

Tenemos una amplia variedad de herramientas de alta calidad, tanto para uso doméstico como profesional. Encuentra desde pinzas hasta herramientas eléctricas y destornilladores.

Plomería

Contamos con todo lo que necesitas para renovar o arreglar tu baño. Tenemos griferías, tuberías, sanitarios y repuestos de todo tipo. Si necesitas solucionar cualquier problema de plomería, estamos aquí para ayudarte.

Electricidad

En Shama encontrarás todo lo necesario para tus instalaciones eléctricas. Ofrecemos una amplia variedad de productos como bombillas LED, cinta de aislar, cable eléctrico, tomacorrientes, productos de calidad para asegurarte una instalación segura y confiable.

Pinturas

Tenemos desde pinturas látex y esmalte brillante, hasta opciones como pinturas anticorrosivas y tintes para madera y cemento. ¡Contamos con los accesorios necesarios para realizar tus proyectos de pintura!
